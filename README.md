Meteor - IBM Connections
===

# Prerequisites

Install Node.js (including Node package manager), Meteor and meteorite.

# Installation

Execute following commands:

`mrt install`

`mrt add accounts-ui`

`mrt remove autopublish`

# Execution

Execute following commands:

`mrt`

# Demo

Go to [http://optis.meteor.com](http://optis.meteor.com)
